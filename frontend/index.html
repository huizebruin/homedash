Externe kalender-feeds
Voeg een Google Calendar of andere iCal-link toe. Events worden weergegeven in de agenda.

Google Calendar public link:
1. Google Calendar → ⚙ Instellingen → klik op kalender
2. Scroll naar "Publiek adres in iCal-formaat"
3. Kopieer de link (begint met webcal:// of https://calendar.google.com/...ical/...)

Nextcloud / andere: gebruik de .ics URL van je agenda-app.

🏠 home//dash

Een open-source homelab browser startpagina. Gemaakt door Wobbe Bruin (huizebruin) met AI-assistentie (Claude door Anthropic).

Toont Docker containers, systeem stats, diensten & IP's, weer, kalender en agenda — allemaal op één scherm.

⭐ GitHub 🐛 Issues 📋 Changelog 📄 MIT License

🔧 Tech stack

Pure HTML/CSS/JS frontend · Node.js API · Docker · Nginx · Open-Meteo weer API
