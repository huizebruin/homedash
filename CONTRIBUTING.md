# Bijdragen aan home//dash

Bedankt voor je interesse! Alle bijdragen zijn welkom.

## Bugs melden

Open een [issue op GitHub](https://github.com/huizebruin/homedash/issues) met:
- Wat je verwachtte te zien
- Wat er werkelijk gebeurde
- Browser + OS
- Docker versie

## Verbeteringen voorstellen

Open ook een issue met het label `enhancement`.

## Pull requests

1. Fork de repository
2. Maak een branch: `git checkout -b feature/mijn-feature`
3. Commit je wijzigingen
4. Open een pull request

## Projectstructuur

- `frontend/index.html` — volledige UI (HTML/CSS/JS, geen framework)
- `api/server.js` — Node.js HTTP server
- `data/config.json` — gebruikersdata (niet committen!)

## Code stijl

- Vanilla JS, geen frameworks
- CSS variables voor theming
- JetBrains Mono voor code/getallen, Inter voor tekst
